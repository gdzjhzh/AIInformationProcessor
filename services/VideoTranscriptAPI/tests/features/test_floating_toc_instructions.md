# 浮动 TOC 功能测试指南

## 测试环境

### 方法一：使用独立测试页面

1. 直接在浏览器中打开测试 HTML 文件：
   ```
   tests/features/test_floating_toc.html
   ```

2. 测试场景：
   - 该页面模拟了实际的转录结果页面结构
   - 包含多层级标题（H1-H4）
   - 包含"内容总结"和"校对文本"两个区块

### 方法二：使用实际 API 服务器

1. 启动 API 服务器：
   ```bash
   python main.py --start
   ```

2. 通过 API 创建一个转录任务，获取结果页面

3. 访问转录结果页面，测试 TOC 功能

## PC 端测试清单

### 基础功能
- [ ] TOC 容器是否正确显示在页面右侧
- [ ] 初始状态是否为收起（显示指示线）
- [ ] 鼠标悬停时是否展开
- [ ] 鼠标移开后是否收起

### 指示线效果
- [ ] 收起状态下是否显示 4 条渐变指示线
- [ ] 指示线宽度是否递减
- [ ] 悬停时指示线是否消失

### Pin 固定功能
- [ ] 点击 Pin 按钮是否固定展开状态
- [ ] 固定后 Pin 图标是否高亮
- [ ] 再次点击是否取消固定
- [ ] Pin 状态是否保存到 localStorage
- [ ] 刷新页面后 Pin 状态是否保持

### TOC 内容
- [ ] 是否正确提取"内容总结"区块的 H1-H4 标题
- [ ] 标题层级缩进是否正确（H1: 16px, H2: 32px, H3: 48px, H4: 64px）
- [ ] 是否显示"✨ 校对文本"链接
- [ ] 标题文本是否完整显示

### 点击跳转
- [ ] 点击标题链接是否平滑滚动到对应位置
- [ ] 点击"校对文本"链接是否跳转到校对文本区块
- [ ] 跳转后是否更新激活状态

### 滚动高亮
- [ ] 滚动页面时是否自动高亮当前标题
- [ ] 高亮样式是否正确（左侧蓝紫色边框，背景色变化）
- [ ] 切换标题时高亮是否平滑过渡

### 主题适配
- [ ] 浅色主题下颜色是否正确
- [ ] 深色主题下颜色是否正确
- [ ] 切换主题时 TOC 样式是否同步更新

### 滚动条
- [ ] TOC 内容过多时是否显示滚动条
- [ ] 滚动条样式是否美观（细窄、半透明）

## 移动端测试清单（屏幕宽度 ≤ 768px）

### 基础功能
- [ ] PC 端 TOC 容器是否隐藏
- [ ] 是否显示右下角浮动按钮（📑 图标）
- [ ] 浮动按钮位置是否正确（距底部 80px，距右侧 20px）

### 浮动按钮
- [ ] 按钮大小是否为 56x56px
- [ ] 按钮是否为圆形
- [ ] 渐变背景色是否正确
- [ ] 阴影效果是否明显
- [ ] 点击时是否有缩放动画

### 弹出面板
- [ ] 点击浮动按钮是否弹出 TOC 面板
- [ ] 面板是否从底部滑入（60vh 高度）
- [ ] 是否显示半透明遮罩
- [ ] 面板头部是否显示"📑 目录"和关闭按钮

### 关闭面板
- [ ] 点击关闭按钮（✕）是否关闭面板
- [ ] 点击遮罩层是否关闭面板
- [ ] 点击 TOC 项后是否自动关闭面板

### TOC 内容
- [ ] TOC 项字体大小是否适合移动端（1rem）
- [ ] 点击区域是否足够大（padding: 12px 16px）
- [ ] 是否支持滚动查看更多内容

### 交互体验
- [ ] 打开面板时页面是否禁止滚动
- [ ] 关闭面板后页面是否恢复滚动
- [ ] 动画是否流畅
