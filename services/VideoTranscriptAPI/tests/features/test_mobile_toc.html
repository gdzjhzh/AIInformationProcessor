移动端 TOC 功能测试

📝 内容总结

第一部分：移动端测试

这是第一部分的内容，用于测试移动端 TOC 功能。请向下滚动查看更多内容。

测试要点：浮动按钮、面板弹出、关闭功能。

1.1 基础功能测试

测试浮动按钮是否显示在右下角。

按钮应该是圆形，带有渐变背景色。

1.1.1 点击测试

点击浮动按钮，应该弹出目录面板。

面板应该从底部滑入，占据屏幕下半部分。

1.1.2 关闭测试

点击 ✕ 按钮，应该关闭面板。

点击遮罩层，也应该关闭面板。

点击目录项后，应该自动关闭面板并跳转。

1.2 交互测试

测试各种交互场景。

1.2.1 滚动测试

滚动页面时，浮动按钮应该保持在固定位置。

1.2.2 主题切换测试

切换主题时，TOC 颜色应该自动适配。

第二部分：深入测试

这是第二部分，继续测试更多功能。

2.1 性能测试

测试打开/关闭面板的流畅度。

2.1.1 动画效果

面板滑入动画应该流畅。

2.1.2 响应速度

按钮点击应该立即响应。

2.2 边界测试

测试各种边界情况。

2.2.1 快速点击

快速多次点击按钮，不应该出现异常。

2.2.2 内容滚动

在面板内滚动目录，应该流畅。

检查所有测试项是否通过。
